◆ hasAllFeats()

bool Node::hasAllFeats ( std::string  value)

Check if the Node's FEATS has all morphological features specified in the string value, e.g.

node->hasAllFeatures("Case=Nom|PronType=Prs")

Note that the check is not exclusive, e.g. consider the node with morphological properties Case=Nom|Person=2|PronType=Prs, then the following holds.

node->hasAllFeatures("Case=Nom|Person=2") // returns true
node->hasAllFeatures("Case=Nom") // returns true
node->hasAllFeatures("Case=Nom|Person=3") // returns false
